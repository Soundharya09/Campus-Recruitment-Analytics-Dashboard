# 🎓 Campus Placement Analytics Dashboard | MySQL + Tableau
## 📌 Project Overview
The Campus Placement Analytics Dashboard is an end-to-end data analytics project designed to analyze student placement performance, salary 
outcomes, and key employability factors using interactive Tableau dashboards powered by a MySQL database.

This project transforms raw student placement data into meaningful insights that help universities, training institutes, and placement cells 
improve hiring outcomes and student career readiness.

## 🎯 Business Problem
University placement cells often struggle to answer critical questions such as:

Q1. What factors influence student placement success?

Q2. How do technical, aptitude, and communication skills impact hiring?

Q3. Which specializations receive the highest salaries?

Q4. Does internship experience improve placement chances?

Q5. What academic performance levels are required for high-paying jobs?

Without structured analytics, placement decisions rely on intuition rather than data-driven insights.

This project solves this problem by building interactive dashboards that reveal placement trends and hiring patterns.

## 🧠 Project Objectives
- Analyze overall campus placement performance
- Identify factors influencing placement probability
- Evaluate salary distribution across students
- Understand the impact of skills and academic scores
- Compare gender-based placement performance
- Analyze specialization and experience impact on salaries
- Build executive-ready placement dashboards

## 🛠 Tech Stack
- Database: MySQL
- Visualization Tool: Tableau
- Data Processing: SQL + Tableau Calculated Fields
- Dataset Type: Student Placement Dataset

## 🔄 End-to-End Workflow
### Data Storage (MySQL)
- Imported student placement dataset into MySQL
- Structured tables for efficient querying
- Enabled scalable data storage for analytics

### Data Extraction
- Connected Tableau directly to MySQL
- Optimized queries for dashboard performance
- Enabled real-time data analysis

### Data Transformation
Created analytical metrics, including:
- Placement Rate (%)
- Average Salary
- Salary Bands
- Skill Score Categories
- Experience Groups
- Academic Score Bands

These calculated fields help identify key patterns affecting placements and salaries.

### Data Visualization
Designed interactive dashboards following data visualization best practices:
- Clear visual hierarchy
- Minimal cognitive load
- Interactive filtering
- Executive-level storytelling

## 📈 Key Insights from the Project
- Students with strong technical and aptitude scores have a higher placement probability
- Internship experience significantly increases salary offers
- Certain specializations receive higher average salary packages
- Academic performance positively correlates with placement success
- Communication skills influence high salary offers

- Guide students toward high-demand career paths
